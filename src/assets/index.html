Keyboard Shortcuts

  • Ctrl+Shift+C - Copy selection
  • Ctrl+Shift+V - Paste from clipboard
  • Ctrl+Shift+F - Search terminal buffer
  • Ctrl+Shift+S - Export terminal buffer to file

File Transfer (ZMODEM)

Upload files to server: rz

Wait for the file picker to appear, select your files, and they will be uploaded to the current directory.

Download files from server: sz <filename>

The server will send the file and your browser will prompt you to download it.
